How to navigate in LangSmith:

  1. Log in to your LangSmith console with your active credentials.
  2. Click on the Projects tab on the left sidebar.
  3. Select the project named auto-data-analysis.
  4. Click on the latest run thread to expand the full agent cascade (Waterfall View). You can click on individual nodes to review the input prompts and output responses for each agent.
